# Publisher Module
[![Build Status](https://travis-ci.org/spryker/publisher.svg)](https://travis-ci.org/spryker/publisher)

Publisher module is a registerer for publisher plugins to Event\Listener.

## Installation

```
composer require spryker/publisher
```
